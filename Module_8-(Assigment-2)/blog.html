VAR:

Variables declared with var have either global scope or function scope. This means that they can be accessed from anywhere within the global scope or from within the function in which they are declared.

LET:

Let Variables declared with let are block-scoped. Let variables are hoisted to the top of their block but not initialized.

CONST:

Const variables must be initialized at the time of declaration, and their values cannot be reassigned. However, for objects and arrays, while the variable itself cannot be reassigned, the properties or elements of the object or array can be modified.

NULL:

Null is often used to explicitly indicate the absence of any object value. It's a deliberate assignment to signify that a variable or property does not point to any valid object or value.

Undefined:

Undefined typically represents the absence of an assigned value in variables. When a variable is declared but not initialized.
